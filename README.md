# **printf-garden** - is a tutorial game for C language, where 'printf()' commands grow into a beautiful garden.
The player study format specifiyers ('%d', '%s', '%f', '%c' and others), growing flowers, trees, bushes, ponds and decorations, using proper printf() commands.
---
